# Интеграции с ИИ-провайдерами

Проект "Плати по миру" — это агрегатор лучших ИИ-технологий. Ниже описано, за что отвечает каждый провайдер и какие ключи нужны.

## 🔗 Список провайдеров

### 1. OpenRouter (LLM)
- **Зачем**: Написание сценариев, экстракция ключевых слов, генерация промптов для видео, механизм AI Evolution.
- **Модели**: `google/gemini-2.0-flash-exp-001`, `anthropic/claude-3.5-sonnet`.
- **Конфигурация**: `OPENROUTER_API_KEY`.

### 2. KIE.ai (Veo-3)
- **Зачем**: Самая современная генерация видео-перебивок (B-roll) по текстовому описанию.
- **Особенности**: Система Veo-3 позволяет добиваться высокой кинематографичности.
- **Конфигурация**: `KIE_API_KEY`, `KIE_BASE_URL`.

### 3. HeyGen
- **Зачем**: Генерация "говорящей головы" (Digital Avatar) с идеальной синхронизацией губ (Lipsync).
- **Конфигурация**: `HEYGEN_API_KEY`.

### 4. ElevenLabs
- **Зачем**: Премиальная озвучка и клонирование голоса.
- **Особенности**: Используется для клиентов, требующих максимального качества речи.
- **Конфигурация**: `ELEVENLABS_API_KEY`.

### 5. MiniMax
- **Зачем**: Качественная и быстрая озвучка (альтернатива ElevenLabs).
- **Особенности**: Очень хорошо справляется с русским языком.
- **Конфигурация**: `MINIMAX_API_KEY`, `MINIMAX_GROUP_ID`.

### 6. Deepgram
- **Зачем**: Молниеносная транскрибация аудио для получения таймингов слов.
- **Конфигурация**: `DEEPGRAM_API_KEY`.

---

## ⚙️ Настройка переменных окружения (.env)

Для запуска всех сервисов убедитесь, что в файле `.env` прописаны следующие ключи:

```bash
# Core
DATABASE_URL=postgresql://...
REDIS_URL=redis://...

# LLM
OPENROUTER_API_KEY=sk-or-...

# Video/Avatar
KIE_API_KEY=...
HEYGEN_API_KEY=...

# Audio
ELEVENLABS_API_KEY=...
MINIMAX_API_KEY=...
MINIMAX_GROUP_ID=...
DEEPGRAM_API_KEY=...

# Storage
YANDEX_DISK_TOKEN=...
```
